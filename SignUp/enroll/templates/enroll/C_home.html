Room's Name
Read the RULES and IMP. POINTS Carefully !

. . . RULES . . .

* 5 Steps to be followed !
  • 1. First character is dollar. ' $ ' .
  • 2. Second character is user's name's First alphabet .
  • 3. Third character is character at position Y. Y is given as Y = ( length of user's name's ) / 2 .
  • 4. Fourth character is user's name's Second Last Alphabet .
  • 5. Fifth character is dollar. ' $ ' .

    • For Eg. Suppose name of person is Rajneesh .
    For making Room's Name, Take all alphabet in CAPITAL LETTER , i.e RAJNEESH . ( Info. 1 )
    Then Start,
    First Character is '$' . ( Rule 1 )
    Second Character is 'R' . ( Rule 2 )
    Third Character is 'N' . ( Rule 3 )
    ∵ Length of RAJNEESH is 8 .
    ∴ Y => 8 / 2 => 4
    ∴ Y = 4
    Third Character will be ' N ' .
    { Note : In Case of odd no. divisibility / 2 , Don't Roundoff & Just ignore the decimal places. }
    Fourth Character is 'S' . ( Rule 4 )
    Fifth Character is '$' . ( Rule 5 )
    So Final Room's Name will be '$RNS$'

. . . IMP. POINTS . . .

  • 1. Users must create Room's name by taking name in CAPITAL LETTER , if not they will be responsible for cause.

  • 2. User 1 should make Room's Name by using User 2's Name & check that Is this room have been created yet or Not ? If Not then he have to create Room's name by using his own name and vice versa.

    User 1 = Ram & User 2 = Bharat
